  ## Description
  This Express.js API uses Sequelize to interact wity a MySQL database. The database keeps track of the different categories, products and tags of an Ecommerce site. 
  
  ## Installation
  - npm i
  - mysql -u root -p 
  - source db/schema.sql
  - exit
  - npm run seed
  - npm start
